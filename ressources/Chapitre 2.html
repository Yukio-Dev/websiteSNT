Chapitre 2 : Comment écrire du code ?
Pour écrire du code, que tu sois depuis un ordinateur Windows ou Mac, tu vas avoir besoin d'un logiciel adapté à ton langage. Même si techniquement, la plupart des langages pourraient s'écrire dans un simple bloc-note, ce n'est pas le plus simple et c'est loin d'être le plus ergonomique.
Quel logiciel choisir pour du :

  • Python: Pour du Python, plusieurs possibilités s'offrent à toi. Les plus connus sont l'IDE PyCharm (IDE signifie "integrated development environment". C'est un logiciel qui regroupe plusieurs outils comme un editeur de texte, un compilateur, une console de débogage etc), Visual Studio Code, et Sublime Text. Tous ces logiciels sont très similaires, à toi de choisir lequel tu préfères en fonction de l'esthétique, de la facilité d'utilisation et des fonctionnalités(options,extensions...)

  • C++ et C#: Les IDE les plus connus pour faire du C++ sont Code::Blocks et Visual++ sur Windows, et XCode sur Mac.
    Pour faire du C# tu peux utiliser l'éditeur de texte Visual Studio Code ainsi que l'IDE Visual Studio (à ne pas confondre)

  • Java: Pour faire du Java, les logiciels les plus connus sont IntelliJ Idea et Netbeans. La plupart des développeurs ont commencé sur Eclipse mais migrent progressivement vers IntelliJ Idea à cause de bugs et de son interface vieillisante.

  • Kotlin et Java (pour le développement mobile uniquement) :
    Pour le développement mobile, l'IDE le plus utilisé est Android Studio. L'outil fait par Google à le monopole dans le milieu des IDEs accés développement mobile. Il est grandement apprécié pour son interface de prévisualisation en direct et son émulateur Android (il vous permet d'utiliser un téléphone Android virtuel afin de tester votre application, sans avoir à l'injecter systematiquement dans votre téléphone physique).

  • Swift: L'éditeur de texte le plus populaire est Atom, il est disponbible sur Windows et Mac. Sublime Text et Visual Studio Code sont aussi utilisés bien que moins courants.

  • HTML: Le langage HTML est construit simplement que n'importe quel éditeur de texte suffit, même un bloc-note. Les éditeurs de textes les plus utilisés sont Sublime Text, mais surtout Visual Studio Code, très pratique grâce à toutes ces extensions faites par la communauté (c'est celui que j'utilise personnellement)
